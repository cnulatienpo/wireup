How numbers become space

This guide explains stateful vs stateless operators TouchDesigner in plain language. It focuses on practical decisions you can make while building and debugging TouchDesigner networks.

Some operators only care about what arrives this frame.

Others remember what happened before.

That difference changes everything.

A stateless operator evaluates its input and produces an output based only on the current moment. It has no memory. If the same input arrives again, the same output is produced.

A stateful operator carries history forward. It accumulates. It delays. It integrates. It compares now to before.

Lag, Feedback, Trail, and certain CHOPs are stateful. A Math node performing simple scaling is not.

This distinction explains why some nodes behave predictably and others drift, build up, or decay over time.

When a node appears to “remember,” it is not thinking. It is storing previous values and using them in current evaluation.

Statefulness introduces accumulation. Accumulation introduces instability if not controlled.

That is why feedback networks can spiral, freeze, or saturate. The system is honoring past values repeatedly.

Resetting a stateful system requires interrupting its memory. That might mean toggling a Reset parameter, clearing a buffer, or briefly breaking a feedback connection.

Stateless systems are easier to reason about because they do not depend on history. Stateful systems are powerful because they create continuity.

Choosing between them is not about difficulty. It is about whether your design depends on persistence.

FAQ

What is a stateful operator in TouchDesigner?

A stateful operator stores and uses information from previous frames, allowing accumulation, delay, or feedback-based behavior.

Why does this node remember previous frames?

Because it maintains internal buffers or feedback loops that carry past values forward into current evaluation.

How do I reset a stateful network?

Use Reset parameters, clear buffers, or temporarily break feedback connections to remove stored history.
